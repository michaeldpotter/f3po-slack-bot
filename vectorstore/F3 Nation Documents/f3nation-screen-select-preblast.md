# Select Preblast Screen

Use this screen to choose what event you’re creating a preblast for.

> Navigation: Help menu → start preblast flow (see `f3nation-help-menu-calendar-preblast.md`)

---

## Option A: Select From Upcoming Qs
At the top you’ll see **Select From Upcoming Qs:** with buttons for upcoming events (examples shown):
- **03/16 Ground Zero Run/Ruck**
- **03/19 Mastodon Bootcamp**

Click one of these to create a preblast for that event.

---

## Option B: Open Calendar
Text shown: **Sign up to Q for an upcoming event from the calendar:**

- Click **Open Calendar**
- Find the event in the calendar and sign up to Q if needed
- Return to create your preblast

---

## Option C: New Unscheduled Event (not on the calendar)
Text shown: **Or, create a preblast for an event not on the calendar:**

- Click **New Unscheduled Event**

Use this only for “off the books” events.

---

## Tips
- If you don’t see your event under **Select From Upcoming Qs**, you may not be signed up as the Q (or the event may be outside the list window).
- For scheduled workouts, prefer **Upcoming Qs** or **Open Calendar** over **New Unscheduled Event**.
