# F3 Nation App Glossary (for Q&A)

Use this glossary to keep terminology consistent across help answers.

---

## Core F3 terms

### PAX
A participant (a person in the F3 community).

### HIM
A “High Impact Man” — common F3 shorthand.

### AO
“Area of Operations.” In practice: a named workout location / community hub (often maps to a Slack channel).

### Q
The leader of a workout/event (“Q-ing” = leading).

### Q Slot / Open Q Slot
A workout event that has space available for a Q to claim/lead.

### HC
“Hard Commit.” A way to commit that you’re attending an event/workout.

### Preblast
A message posted before a workout to share details like what/where/when and motivate PAX to show up.

### Backblast
A post-workout recap (attendance + notes/moleskin) submitted after the event.

### Moleskin
The notes/summary portion of a backblast.

### Downrange
Traveling / visiting another region. “Downrange PAX” refers to a visitor who may not be in your local Slack.

---

## App + Slack terms

### Workspace
A Slack workspace (your region’s Slack).

### Region
Your F3 region (e.g., Wheaton). Some features depend on being connected to a region.

### Event
A scheduled workout entry (has date/time, AO, event type, and actions like HC/Q/preblast/backblast).

### Event Type
Category of event (e.g., Bootcamp, Run/Ruck). Used in filters and reporting.

### Thread
A Slack message thread. Best practice for Q&A is to keep troubleshooting conversations in one thread.

---

## Screens / menus referenced in this context folder

### F3 Nation Settings
The hub screen with buttons like **User Settings**, **Help Menu**, **Emergency Info Access**, etc.

### Help Menu
A “how-to” menu for common actions (calendar, preblast, etc.). Often the easiest starting point for new users.

### Calendar Home
The “Upcoming Schedule” view with filters (AOs, Event Types, Start Search Date) and checkboxes like:
- Show only open Q slots
- Show only my events

### Select Preblast
The screen where you choose:
- **Select From Upcoming Qs** (pick one of your upcoming Q events)
- **Open Calendar** (go sign up to Q from the calendar)
- **New Unscheduled Event** (only for events not on the calendar)

---

## Admin / advanced terms you may see

### SLT
“Site Leadership Team” (or region leadership). Settings may differ by permissions.

### Paxminer / Slackblast / QSignups (Legacy Tools)
Older tools many regions used before migrating to the F3 Nation app ecosystem.

### Migration Date
The cutoff date when a region switches systems (what’s “before” vs “after” may affect where data is written and how signups work).

### Paxminer Mapping
A tool to map old/legacy channels or data into the new system’s AOs / event types.

### PAX Vault
A reporting/dashboard site used for stats and trends (often referenced for analytics instead of custom reports).

---

## Common “Which feature do I use?” quick answers

- “I want to claim a Q slot” → Calendar Home → show open Q slots → take Q
- “I want to post a preblast” → Select Preblast → choose upcoming Q → fill form
- “I want to update my profile/home region” → User Settings → Submit
- “I need emergency info” → Emergency Info Access (genuine emergencies only)
