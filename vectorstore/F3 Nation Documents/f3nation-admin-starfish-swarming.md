# **📝 Starfish/Swarming Instructions**

**Purpose:**  
 This SOP provides a process to follow when a region currently using the nation app is ready to starfish.

**Platform:**  
 F3 Nation Slack App

**Credits:**  
 Thanks to Bamm-Bamm (Vanguard AZ) for paving the way and documenting his experience for this guide**.**

---

## **🧭 Getting Started**

### **1\. Plan launch date for new slack workspace.**

### **2\. Create new region in [map.f3nation.com/admin](http://map.f3nation.com/admin)**

* You will need to contact your Sector Expansion Q to complete this step.

### **3\. Create a [new slack workspace](https://slack.com/help/articles/206845317-Create-a-Slack-workspace)**

* You can create all of your slack channels, add emojis, etc. at this time.

### **4\. Install F3 Nation App and follow steps 1-9 and 11**

* On Step 7, you will choose today’s date for the migration date.  
* This will connect your slackspace to the new region to allow you to begin tweaking settings in the nation app.  
* This has no impact on the parent region, as the current region will not yet have any AOs/locations/events associated with it.

### **5\. Invite PAX to join the new slack workspace**

* The timing for this is up to you, but this should happen before any AOs are moved from the parent region to ensure your users Q calendar data transfers.  
* All users must use the same email address they use in the parent workspace, as their user data in the nation app is associated with that email address.   
* **NOTE: It is possible that Q Calendar data will not transfer.  Inform PAX that they should take note of their upcoming Qs, so they can reclaim them, if necessary.**

### **6\. All of the above steps should be completed before moving on to the next section.**

## **🧭 The Launch**

### **1\. Have PAX whose home AO will be in the new region update their home region in the User Settings of the Nation App.**

* You can find instructions for this in the [FAQ](?tab=t.5anzgzo1iu9) of this document or watch this [video](https://youtu.be/aP-mghe3jEg?si=cY5XdRadmxYpXJob).

### **2A. If you desire to have Calendar data backed up, the backup should be completed before proceeding with 2B.** 

### **2B. Edit each AO in the [map admin](http://map.f3nation.com/admin), reassigning its parent region to the new region.**

* Documentation for working with F3 Nation Maps can be found [here](http://mapinstructions.f3nation.com/).

### **3\. Assign AOs to Slack channels or set preblast/backblast channels.**

* Details can be found in step 10 of [Installing F3 Nation](?tab=t.8izebhckznu8) in this guide.
