# Emergency Info Access Screen

This screen allows searching for a user’s emergency info.

⚠️ Important warning shown on screen:
- The user will be emailed to let them know their emergency information was accessed.
- Only use this feature in genuine emergency situations.

> Navigation: Settings Home → **Emergency Info Access** (see `f3nation-screen-settings-home.md`)

---

## Option 1: Local Slack User
Search within your current Slack workspace.

- Field: **Select Local User (optional)**
- Dropdown text: “Select a user from this workspace”

Use this when the person is in your workspace.

---

## Option 2: Downrange User Search
Search for users from other regions who have opted into downrange sharing.

- Field: **Search External User (optional)**
- Input guidance: “Type to search for a user by F3 name”

Use this when the person is in another region/workspace and has downrange sharing enabled.

---

## Tips / FAQ

### “Why can’t I find someone?”
Common causes:
- They are not in your local workspace (try Downrange search)
- They have not enabled downrange sharing (their User Settings)
- You may not have permission to access this screen

### “Will the user know?”
Yes. The screen explicitly states the user will be emailed when their emergency info is accessed.
