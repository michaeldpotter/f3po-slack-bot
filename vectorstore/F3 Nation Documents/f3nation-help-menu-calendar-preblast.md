# Welcome to the F3 Nation help menu!

If you're looking for a full guide, see the start of our manual here. Otherwise, see below on how to find the calendar, sign up to Q, and send out your first preblast!

## How to find and use the calendar

You can get to your region's calendar of workouts by starting to type `/` and then either:

- use the `/f3-calendar` command, or
- select the **Open F3 Calendar** shortcut.

There will also be buttons to find the menu, like the one above!

Once in the calendar view, you can see upcoming events listed. You can filter by date, status, AO, etc.

You can interact with events by clicking the overflow menu button next to an event:
- `...` on desktop
- **More** on mobile

From there you can:
- Hard Commit (**HC**)
- sign up to Q (if it’s open)
- view the preblast

---

## How to post a preblast

To post a preblast, there are several ways to start:

1. Start typing `/`, select the `/preblast` command, and hit send — or select the **Create Preblast** shortcut.
2. Click the **Create Preblast** button in the reminder message you’ll get before the event.
3. In the calendar view, click the overflow button on the event (`...` on desktop / **More** on mobile), then choose **Edit Preblast**.

When you use the first option, you can choose between:
- upcoming events you're signed up to lead, or
- posting one for an unscheduled event.

The unscheduled option should only be used for “off the books” events.

After filling out the preblast form, you can choose whether to:
- post it immediately to Slack, or
- wait for the automated preblast to be sent (usually ~12 hours before the event).

Once it is posted, you can edit it by using its **Edit** button or through the calendar menu.

If you do not fill out a preblast by the automated time, a generic one will be posted for you.
