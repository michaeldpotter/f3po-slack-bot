# Calendar Home Screen

The **Calendar Home** screen shows the upcoming workout schedule and lets you filter events and take actions on an event.

> Navigation: Settings/Help → Open Calendar (see `f3nation-help-menu-calendar-preblast.md`)

---

## Upcoming Schedule filters

### Filter AOs (optional)
Use **Filter AOs** to narrow the list to specific AOs.

### Filter Event Types (optional)
Use **Filter Event Types** to narrow by event/workout type.

### Start Search Date (optional)
Use **Start Search Date** to start the list from a certain date.

### Other options (optional)
- ✅ **Show only open Q slots**
- ✅ **Show only my events**

---

## Reading the schedule list

Events are grouped by date (example shown: **Monday, March 09**).

An event line may include:
- Event name (example: **Tar Pit Bootcamp**)
- Time (example: **@ 0530**)
- Q / leader (example: **Q: T-bone**)
- A small icon next to the event (may indicate preblast/edit availability)
- An overflow menu button **(...)** at the far right

---

## Taking action on an event

### Open the event menu
Click the **(...)** button on the event row.

From there, you may be able to:
- **Hard Commit (HC)**
- **Sign up to Q** (if there is an open Q slot)
- **View Preblast**
- **Edit Preblast** (if available)

(Exact options depend on the event state and your permissions.)

---

## Troubleshooting

### “I can’t find the event”
Try:
- clearing AO/Event Type filters
- setting **Start Search Date**
- unchecking **Show only my events**

### “I only see a few events”
You may have filters turned on, or the Start Search Date is limiting the results.
